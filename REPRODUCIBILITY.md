# Reproducibility Guide

## How to Reproduce This Experiment

### Prerequisites
- Access to LM Arena or similar platform
- Accounts for each model provider (if required)

### Steps
1. Navigate to LM Arena (or your chosen platform)
2. Select the target model
3. Copy the exact prompt from this repository
4. Send the prompt **once**
5. Record the complete response
6. Repeat for all models listed

### Important Notes
- Results may vary slightly due to:
  - Model updates after the test date
  - Platform-specific configurations
  - Randomness in generation (even at default settings)
- **Recommendation:** Run the experiment multiple times and average the results for more reliable findings.

### Contribution
If you reproduce this experiment, please:
- Open an issue or pull request with your results
- Mention the date and platform used
- Note any differences in model versions
